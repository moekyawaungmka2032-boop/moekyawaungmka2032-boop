## Deployment

This site auto-deploys to GitHub Pages on every push to `main`.

```yaml
# .github/workflows/deploy.yml (example)
on:
  push:
    branches: [main]
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/deploy-pages@v4
```
